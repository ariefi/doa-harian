# Doa Harian

Website statis berisi kumpulan doa harian lengkap dengan tulisan Arab dan arti.  
Dapat dibuka melalui browser maupun di-install sebagai aplikasi (Progressive Web App).

## Live
https://ariefi.github.io/doa-harian/

## Features
- Dzikir & doa setelah sholat
- Tulisan Arab + arti
- Mode Arabic only
- Installable (PWA)
- Offline support
